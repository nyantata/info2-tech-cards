たとえると

散布図の点の並びに、全体の傾向を表す直線を引くイメージです。直線と実際の点のずれを残差といいます。

目的変数 y

説明・予測したい値。

説明変数 x

目的変数を説明するために使う値。

回帰係数

xが1単位増えたとき、yの予測値が平均でどれだけ変わるかを表す数値。

切片

x=0のときの予測値。ただし、実際にx=0があり得るかを確認して読みます。

残差

観測値−予測値。

R²

予測の直線が、yの変化をどのくらい説明できているかを表す数値。1に近いほど必ず良いとは限りません。

CONCEPT GUIDE

回帰直線だけでなく残差と評価を読む

まず図と要点を確認します。さらに知りたい内容は、見出しをクリックすると下に開きます。

使う場面

2つの量の関係を直線でおおまかに表し、傾向や予測値を調べるとき。

気をつけること

2つの量に関係があるだけで、原因と結果だと決めつけないこと。集めたデータの範囲を超えて予測したり、R²だけで良し悪しを決めたりしないこと。

観測値・予測値・残差

残差 = 観測値 − 予測値です。残差が曲線のように並ぶ場合や、xが大きくなるほど残差が広がる場合は、直線だけでは関係を十分に表せていない可能性があります。

R²は負になることがある
R² = 0 は、テストデータの平均値を常に予測する基準と同程度です。R² < 0 は、その基準より予測誤差が大きいことを示します。計算エラーとは限りません。

データ数が少ない場合や、1回だけデータを分けた場合は、評価が偶然に左右されやすくなります。データの分け方を変えて何度か評価する「交差検証」でも確かめます。

係数を単位付きで読む

適切な読み方

「Xが1単位大きいとき、Yの予測値は平均で回帰係数の分だけ変化する関係が見られた」

言い過ぎ

「Xを増やせば必ずYが増える」。観察データの回帰だけでは因果を証明できません。

REVIEWED MINIMUM EXAMPLE
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score, mean_squared_error

df = pd.read_csv("assets/data/SSDSE-E-2026.csv", encoding="cp932", header=2)
df = df[df["都道府県"] != "全国"].copy()
X = df[["1人当たり県民所得（平成27年基準）"]]
y = df["1人1日当たりの排出量"]

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42
)
model = LinearRegression().fit(X_train, y_train)
pred = model.predict(X_test)
print("回帰係数:", model.coef_[0])
test_r2 = r2_score(y_test, pred)
print("テストデータR²:", test_r2)
if test_r2 < 0:
    print("平均値だけを予測する基準より誤差が大きい結果です。")
print("テストデータRMSE:", mean_squared_error(y_test, pred) ** 0.5)

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。
