START FROM THE BASICS

数式の前に、6つの言葉をつかむ

重回帰分析は、複数の説明変数を足し合わせる形で、1つの目的変数を説明・予測する方法です。最初に、式に登場する記号と言葉を具体例で確認します。

y

目的変数

説明したい、または予測したい値。実際に観測された値を y と表します。

x

説明変数

目的変数を説明・予測する手がかり。複数あるので x₁、x₂、…と表します。

b

回帰係数

ほかの条件が同じだと考えたとき、その説明変数が1単位増えると、予測値がどれだけ変わるかを表します。

b₀

切片

すべての説明変数が0のときの予測値です。0が実際に意味を持つかは、変数の範囲や単位によって異なります。

ŷ

予測値

回帰式を使ってモデルが計算した値です。実際の観測値 y とは区別します。

e

残差

観測値と予測値の差で、残差 = y − ŷ と表します。

SIMPLE VS MULTIPLE

単回帰と重回帰の違い

1単回帰分析

1つの説明変数から、1つの目的変数を説明・予測します。

ŷ = b₀ + b₁x₁

複重回帰分析

複数の説明変数から、1つの目的変数を説明・予測します。

ŷ = b₀ + b₁x₁ + … + bₚxₚ

注意：説明変数が増えるほど、自動的に良いモデルになるわけではありません。モデル作成に使っていないデータで評価します。

気をつけること

説明変数を増やせば必ず良い予測になるとは限りません。係数が大きいだけで、原因や最重要の要因だと決めつけないことも大切です。

予測式と係数の意味

各係数は、「ほかの条件が同じだと考えたとき」の予測値の変化を表します。似た動きをする説明変数が複数あると、どの変数の影響かを分けにくくなり、係数が大きく変わることがあります。

標準化したモデルの係数に注意
コードでStandardScalerを使うと、係数は元の単位ではなく、標準化後の1段階分に対する変化を表します。「1千円増えると…」のように元の単位で読みたい場合は、標準化しないモデルの係数を別に確認するか、元の単位へ戻して計算します。

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

# 公式CSVを読み込む
# Colabではファイルをアップロードして同じフォルダに置く
df = pd.read_csv("SSDSE-E-2026.csv", encoding="cp932", header=2)
df = df[df["都道府県"] != "全国"].copy()

# 単位を確認して派生変数を作る
df["高齢化率"] = df["65歳以上人口"] / df["総人口"] * 100
df["人口密度"] = df["総人口"] / (
    df["総面積（北方地域及び竹島を除く）"] / 100  # ha → km²
)

features = [
    "1人当たり県民所得（平成27年基準）",
    "ごみのリサイクル率",
    "高齢化率",
    "人口密度"
]
X = df[features]
y = df["1人1日当たりの排出量"]

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42
)
model = make_pipeline(StandardScaler(), LinearRegression())
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

print("テストデータR²:", r2_score(y_test, y_pred))
print("テストデータRMSE:", mean_squared_error(y_test, y_pred) ** 0.5)
公式CSV公式解説演習用派生CSV

演習用派生CSVはSSDSE-E-2026から作成したものです。高齢化率と人口密度は上記の式で算出し、列名・単位・年次を明示しています。StandardScalerを含むモデルの内部係数は標準化後の尺度に対する値なので、元の単位の係数として読まないでください。
