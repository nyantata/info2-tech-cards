PROCESS

手順に分けて進める

  1. 1目的と完成条件を書く
  2. 2入力してよい情報か確認する
  3. 3出典・前提・不確実性も出すよう依頼する
  4. 4コードを実行し説明を一次資料と照合する
  5. 5採用・修正・不採用の理由をログへ残す

HANDS-ON

授業内ハンズオン

  1. STEP 1

    曖昧な依頼文と条件付き依頼文を比較する

  2. STEP 2

    生成コードの列名を公式CSVと照合する

  3. STEP 3

    同じ依頼を複数回実行し出力差を記録する

  4. STEP 4

    採用・修正・不採用を利用ログへ残す

CHECKPOINT

CODE / COLAB

最小コードを動かす

REVIEWED MINIMUM EXAMPLE
prompt = """
目的: SSDSE-Eの2変数を散布図にする
条件: cp932、header=2、全国行を除外
依頼: pandasとmatplotlibのコード案を示す
必須: 前提、確認すべき列名・単位、誤りの可能性も書く
禁止: 実在しない列名や出典を作らない
"""
print(prompt)

# 生成されたコードはそのまま採用せず、
# 公式CSVの列名確認と実行テストを行う。

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。

CODE WALKTHROUGH

このコードは、どう動く？

目的・条件・禁止事項を含む依頼文を組み立て、生成結果を公式資料と実行結果で検証するためのコードです。 コード全体を一度に読むのではなく、入力がどのように変化して出力になるかを順番に追います。

DATA / DESIGN MISSION

SSDSEまたはe-Statの公式資料を先に確認し、その列名・単位・年次をプロンプトへ含めます。生成コードをColabで実行し、結果を手計算や別方法で部分確認します。
