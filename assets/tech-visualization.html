CONCEPT GUIDE

グラフを「選ぶ・読む・疑う」

まず図と要点を確認します。さらに知りたい内容は、見出しをクリックすると下に開きます。

使う場面

値がどのあたりに集まっているか、時間による変化、2つの量の関係、グループごとの差を見たいとき。

気をつけること

軸の範囲や、数値を区切る幅を変えると、同じデータでも見え方が変わります。散布図だけで原因と結果を決めつけません。

箱ひげ図を図で読む

箱の左端が第1四分位数、中央の線が中央値、右端が第3四分位数です。箱は、中央にある50%のデータの範囲を表します。赤い点は、ほかの値から大きく離れた値の例です。ひげの端は、外れ値を除いた範囲の端を示しています。

読み取り例：箱が長いほど、中央50%の値が広い範囲に散らばっています。中央値の線が箱の中央からずれていると、値の広がり方が左右で異なる可能性があります。

CODE / COLAB

最小コードを動かす

REVIEWED MINIMUM EXAMPLE
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("assets/data/SSDSE-E-2026.csv", encoding="cp932", header=2)
df = df[df["都道府県"] != "全国"].copy()

plt.hist(df["1人1日当たりの排出量"], bins=8, edgecolor="black")
plt.xlabel("1人1日当たりのごみ排出量（g/人日）")
plt.ylabel("都道府県数")
plt.title("都道府県別の分布")
plt.show()

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。
