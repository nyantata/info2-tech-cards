CONCEPT GUIDE

アンケート結果の前に「誰にどう聞いたか」を確認する

まず図と要点を確認します。さらに知りたい内容は、見出しをクリックすると下に開きます。

使う場面

意識、評価、経験など、センサでは直接得にくい情報を質問によって集めるとき。

気をつけること

回答した人だけの結果を、対象者全体の意見だと決めつけないこと。答えを誘導する質問、2つの内容を一度に聞く質問、少人数から個人が分かってしまう集計にも注意します。

母集団と標本

実際に回答した人たちが、知りたい対象全体と大きく違うと、計算が正しくても結論は偏ります。誰を対象にし、どのように選び、何人が回答したかを記録します。

良い質問と直すべき質問

誘導質問

「この分かりやすい画面をまた使いたいですか」

中立に直す

「この画面をまた使いたいと思いますか」

二重質問

「見やすく操作しやすかったですか」

2問に分ける

見やすさと操作しやすさを別々に尋ねる。

回答率だけでなく偏りを見る
確認項目例
無回答未回答者に共通する特徴がないか
選択バイアス希望者だけが回答していないか
質問の聞き方による偏り質問文や回答する場所・状況が答えへ影響していないか

PROCESS

手順に分けて進める

  1. 1調査目的と利用目的を書く
  2. 2知りたい対象全体と、実際に回答してもらう人の選び方を決める
  3. 31問1内容で中立な質問を書く
  4. 4少人数で試行し誤解を確認する
  5. 5回答率、未回答の項目、回答者の偏りを確認して集計する

HANDS-ON

授業内ハンズオン

  1. STEP 1

    調査目的を「誰の何を知るか」で書く

  2. STEP 2

    誘導質問と二重質問をそれぞれ1問修正する

  3. STEP 3

    5人程度で試行し、迷った表現を記録する

  4. STEP 4

    回答率と無回答の件数を計算する

CHECKPOINT

CODE / COLAB

最小コードを動かす

REVIEWED MINIMUM EXAMPLE
import pandas as pd

df = pd.read_csv("assets/data/survey_demo_simulated.csv")
print(df["ease_of_use"].value_counts(dropna=False).sort_index())
print("有効回答数:", df["ease_of_use"].notna().sum())

# 属性とのクロス集計。少人数のセルは個人特定に注意。
print(pd.crosstab(df["grade"], df["ease_of_use"], normalize="index"))

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。

授業内で必要な最小限の質問を作り、利用目的・保存期間・閲覧者を明示します。氏名など分析に不要な個人情報は収集しません。

解釈と安全の注意

個人情報を直接入力させる場合は利用目的の明示が必要です。人数が少ないグループで複数の情報を組み合わせると、誰の回答か推測できる場合があります。結果を細かく分けすぎないようにします。
