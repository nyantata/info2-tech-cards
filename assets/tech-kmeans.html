使う場面

正解となる分類名がないデータを、似た特徴をもつグループへ分けて、どのようなまとまりがあるか調べるとき。

気をつけること

できたグループ分けを唯一の正解や順位だと考えないこと。使う変数、単位や数値の大きさ、グループ数k、最初の中心の位置で結果が変わります。

中心点が動く4段階

各点を近い中心へ割り当て、割り当てられた点の平均へ中心を移します。グループ分けが変わらなくなるまで、この操作を繰り返します。

標準化の有無で距離が変わる

元の単位

所得が数千、割合が数十のように数値の大きさが違うと、数値が大きい所得のほうが距離へ強く影響します。

標準化後

各変数の平均を0、標準偏差を1にそろえると、単位や数値の大きさによる影響を小さくできます。

k-meansが苦手な形

k-meansは各クラスタの中心からの距離を小さくするため、丸い形で同程度の広がりをもつクラスタに向きます。細長い形、輪の形、点の混み具合やグループの大きさが大きく違うデータでは、不自然な分け方になることがあります。

CODE / COLAB

最小コードを動かす

REVIEWED MINIMUM EXAMPLE
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans

df = pd.read_csv("SSDSE-E-2026.csv", encoding="cp932", header=2)
df = df[df["都道府県"] != "全国"].copy()
features = ["1人当たり県民所得（平成27年基準）", "ごみのリサイクル率"]
X = df[features].dropna()
Z = StandardScaler().fit_transform(X)

model = KMeans(n_clusters=3, random_state=42, n_init=20)
labels = model.fit_predict(Z)
result = df.loc[X.index, ["都道府県"]].copy()
result["cluster"] = labels
print(result.sort_values("cluster"))

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。
