CODE / COLAB

最小コードを動かす

REVIEWED MINIMUM EXAMPLE
import json
from datetime import datetime

log = {
    "time": datetime.now().isoformat(timespec="minutes"),
    "task": "SSDSE-Eを47都道府県に整形",
    "evidence": "notebook_v03.ipynb / 47 rows",
    "decision": "総数ではなく人口当たり指標を使う",
    "issue": "項目年次の確認が未完了",
    "next_action": "解説PDFの項目表を確認する"
}
print(json.dumps(log, ensure_ascii=False, indent=2))

コードは学習用の最小例です。ファイル名、列名、単位、欠損、ライブラリのバージョンを確認して実行してください。

CODE WALKTHROUGH

このコードは、どう動く？

作業内容・記録・判断・課題・次の行動を辞書にまとめ、JSON形式で読みやすく出力するコードです。 コード全体を一度に読むのではなく、入力がどのように変化して出力になるかを順番に追います。

DATA / DESIGN MISSION

班で1つの開発ログを毎回残す

Googleフォームや共有シートで、フェーズ、使用カード、成果、記録、課題、AI利用、次の行動を記録します。個人の評価ではなく開発の再現と支援に使います。

開発ログ雛形確認先

解釈と安全の注意

「頑張った」「進んだ」だけでは再現できません。具体的なファイル、件数、条件、変更理由を残し、個人情報や認証情報は記録しません。
